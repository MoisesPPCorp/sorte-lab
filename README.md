# 🎲 SorteLab

Aplicativo web para **sorteio de times e grupos**, desenvolvido com Node.js, Express, MySQL e JavaScript.

---

# 🚀 Funcionalidades

* Login de administrador
* Cadastro de participantes
* Remoção de participantes
* Lista de participantes
* Sorteio automático de grupos
* Arrastar jogadores entre grupos
* Novo sorteio rápido
* Configuração de tipos de sorteio

---

# 🧠 Tipos de sorteio

O sistema suporta diferentes modalidades:

### ⚽ Futebol

* distribuição automática
* goleiro por grupo
* equilíbrio de níveis

### 🏐 Vôlei

* sorteio de times
* equilíbrio de níveis

### 🏀 Basquete

* distribuição automática
* equilíbrio de níveis

### 🃏 Truco

* modo dupla
* modo trio

### 🏆 Campeonato

* sorteio de grupos

# ⚙ Como executar o projeto

## 1️⃣ Clonar repositório

git clone https://github.com/MoisesPPCorp/sorte-lab.git

---

## 2️⃣ Entrar no backend

cd sorte-lab/backend

---

## 3️⃣ Instalar dependências

npm install

---

## 4️⃣ Configurar banco MySQL

Criar banco chamado:

sorteador_times

Editar conexão no arquivo:

db.js

---

## 5️⃣ Iniciar servidor

node server.js

Servidor rodará em:

http://localhost:3000

---

## 6️⃣ Abrir o frontend

Abrir no navegador:

frontend/index.html
